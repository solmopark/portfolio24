  • 코딩을 시작한 계기는?

    학창 시절부터 컴퓨터에 관심이 많았고 과제를 하더라도 ppt를 도맡았지만 특별히 무엇을 좋아하는지는 정확히 알지 못했습니다. 우연히 코딩을 접했을 때 드는 생각이 디자인 작업은 끝이 없는 미로를 헤매는 것 같았다면 코딩은 코드 실행 결과물이 눈으로 바로 보이는 점에서 흥미롭고 매력적으로 다가왔습니다. 구현한 결과물이 실제 사용되는 것을 보고 나중에 나도 저런 결과물을 만들 수 있으면 좋겠다고 생각했습니다.

  • 어떤 업무를 했었나요?

    ① 웹진
    : (사보 완성 후) 각 페이지별 디자인 확인 ➔ 사보 디자인 맞추어 웹진 디자인 후 웹페이지 코딩 및 반응형 코딩(CSS 및 JS 추가) ➔ 내부 컨펌 ➔ 기관 컨펌
    ② 뉴스레터
    : 웹진 최종 컨펌 후 해당 메인 비주얼 반영하여 디자인 / 메인이미지 없는 경우 클립아트로 계절감 맞춰 디자인 ➔ 시안 컨펌 요청 ➔ 컨펌 후 이미지 업로드. 이미지 절대경로 사용하여 코딩 ➔ 기관 테스트 메일 발송 ➔ 타스온 최종 발송 or 예약 발송
    ③ 카드뉴스
    : 웹진 원고 전체 컨펌 후, 카드뉴스 원고 전달 받음 ➔ 작업 후 내부 컨펌 ➔ 컨텐츠팀 컨펌 ➔ 기관 송부 및 컨펌 요청 ➔ 웹진 게시
    ④ 이벤트
    : 기획안 작성 후 기관 컨펌 요청 ➔ 해당 기획안 디자인(웹게시용, sns게시용) ➔ 이벤트 폼 작성 ➔ 웹진 게시 및 sns 게시 요청
    ⑤ 지결서
    : 각 과업별 발송하는 계정을 매월 체크하여 미리 충전 요청(과업별 타스온/카카오 로그인 정보 다름)
